# Анализ качества вина (Wine Quality Analysis)

Практикум по анализу данных | Студент: Михельсон В.В. | Декабрь 2025

## О проекте

Репозиторий содержит результаты статистического анализа датасета с физико-химическими характеристиками красного вина (1599 образцов). Цель работы — выявление взаимосвязей между параметрами вина и его качеством, проверка статистических гипотез, а также сравнение реализации методов на Python и R.

## Содержание

- `winequality-red.csv` — исходный датасет
- `wine_analysis_PY1.ipynb` — анализ распределений и визуализация на Python
- `wine_analysis_PY2.ipynb` — статистические тесты, корреляция, регрессия, проверка мультиколлинеарности на Python
- `wine_analysis_R1.ipynb` и `wine_analysis_R2.ipynb` — аналогичный анализ на R
